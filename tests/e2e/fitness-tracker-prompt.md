Build a fitness tracking API with the following features:

1. Workouts - Track workout sessions with type (running, weightlifting, cycling, swimming, other), date, duration in minutes, calories burned, notes, and a status (planned, completed, skipped).

2. Running Logs - Track running sessions with distance in miles, duration in minutes, pace, route name, elevation gain, weather conditions, and heart rate average.

3. Exercises - Catalog of exercises with name, muscle group (chest, back, legs, shoulders, arms, core), equipment needed, and difficulty level (beginner, intermediate, advanced).

4. Workout Exercises - Link exercises to workouts with sets, reps, weight, and rest time between sets. References both workout and exercise entities.

5. Progress Metrics - Track body measurements over time: weight, body fat percentage, resting heart rate, and custom metrics. Each entry has a date and value.

Use MongoDB with Docker for testing. Native MongoDB driver.
